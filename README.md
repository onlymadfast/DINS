# Getting Started
#### BUILD WITH JAVA 14.
#### To start:



- [Download this .jar from Yandex.Disk](https://disk.yandex.ru/d/NyLiYe-IdMsA8A?w=1)
- open cmd, then run the jar using: (java -jar tsipadan-1.0.jar)
- must be typed in the browser: http://localhost:8080/swagger-ui.html
